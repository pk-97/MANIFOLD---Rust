Three heights

Collapsed = name bar. Compact = identity + mix (default). Expanded = + clips + routing. The fold ▶/▼ reveals routing; routing never floods the lane.

One grammar

Every type — text/video/gen/group/audio — shares the same skeleton. Type badge (T/▦/◇/▤/∿) carries identity; the blend slot becomes Gain on audio.

Routing form

Folder · MIDI · Channel · Device as a label/value grid. Labels in a fixed column; values aligned, one control vocabulary (all dropdowns ▾).

Selection

Headers carry the solid identity colour (Ableton), so selection can't be a colour fill. Selected layer = bright focus ring + lift (Plasma); selected clip = crisp border (Flowers A).
