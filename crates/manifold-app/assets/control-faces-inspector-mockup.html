How the params map

Same inspector, same params. Faces fold related params into one tool; scalar params stay as sliders. Nothing is lost — every value is still reachable.

Text card

Pos X · Pos Y · Scale → XY pad
H Align · V Align → 3×3 anchor grid
Size · Letter · Line → stay sliders
Text · Font → pickers (footer)

Stylized Feedback

Zoom · Rotate → transform gizmo
Amount → stays slider
Rate · wave · invert → shared editor

Edge Stretch

Direction → compass dial
Width → edge-band frame
Amount · Width → also sliders

The one big change

The rate/waveform block that lived inside the Feedback card is gone. It's now one docked editor at the bottom that retargets to whatever you arm — a slider's ∿, a mod-rail chip, or a gizmo handle.

Mod rail

Auto-built from each widget's bindings — one chip per param, the single place to arm modulation. Border carries source colour: LFO MOD audio.
