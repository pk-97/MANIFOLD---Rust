Sliders unchanged

The default widget keeps its inline T//A row and the same single click. Zero regression for the common case — modulation as a control affordance is just a capability every widget shares.

The mod rail

Auto-generated from a widget's bind map — one chip per param. The single, always-there location to arm modulation. Chip border carries the source colour: LFO MOD audio.

One shared editor

Rate · straight/dotted/triplet · waveform · draw-curve · bipolar depth — one drawer that retargets to whichever param is armed. No four-drawer pileup on multi-param cards.
